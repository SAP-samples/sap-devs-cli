## Use ABAP Test Cockpit before committing
Tags: abap,quality
Run ATC checks on your objects before transporting. Findings block clean-core compliance.

## Prefer CDS views over SELECT from tables
Tags: abap,cds,performance
Define CDS views for all data access — enables authorisation checks, associations, and VDM compliance automatically.

## Use released APIs only in ABAP Cloud
Tags: abap,clean-core
Check the API state in ADT before using any SAP object. Only use APIs with release state "Released for Key User" or higher.

## Enable strict mode in ABAP Cloud
Tags: abap,cloud
Add `abap_cloud` in the ABAP environment to enable strict syntax checks that enforce clean-core rules at development time.
